Cybersecurity Fundamentals Project

A project focusing on the exploration of web application vulnerabilities and brute force attack techniques.

Ready to Start?

Selected Attack Techniques

  • Web Application Vulnerabilities: Exploit Public-Facing Application

    • This attack technique attempts to exploit a vulnerability in an internet-accessible host or system to gain initial network access.

  • Brute Force: Credential Stuffing

    • This technique involves using credentials obtained from data leaks to gain unauthorized access to target accounts.

Project Implementation Plan

    1. Identify issues, select tools, create a GitHub repository, and set up a login form and database.

    • Problem analysis and tool selection. (Betsina, Markowski, Anczarska)
    • Create GitHub repository. (Betsina)
    • Develop login form web page. (Betsina)
    • Database setup for storing login information. (Anczarska)

    2. Write SQL commands to retrieve login details through the authentication form and test server responses.

    • Write SQL command to retrieve login information through authentication form. (Markowski)
    • Use Command Line to retrieve server response - HTML code or Network tab. (Markowski)

    3. Conduct an SQL Injection attack to retrieve logins, analyze the data, and prepare it for further use.

    • Conduct SQL Injection attack to extract login list from the database. (Anczarska)
    • Analyze the results and extract login data for further use. (Anczarska)

    4. Download an external list of frequently used passwords.

    • Download external list of commonly used passwords. (Markowski)

    5. Configure and execute a Brute Force attack with the retrieved logins and password list, documenting and analyzing the results.

    • Configure tool for Brute Force attack using extracted logins and password list. (Markowski)
    • Conduct Brute Force attack and record results (whether login was successful using common passwords). (Markowski, Anczarska, Betsina)
    • Document attack results and analyze effectiveness. (Markowski, Anczarska, Betsina)
